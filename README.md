## Tech Stack

**Client:** Angular, Angular Material, SCSS

**Server:** Node, Express

## Run Locally

Clone the project

```bash
  git clone https://github.com/Jaumoso/pogodex
```

### Frontend

Go to the frontend directory

```bash
  cd pogodex/frontend
```

Install dependencies

```bash
  yarn install
```

Start the frontend

```bash
  yarn start:dev
```

### Backend

Go to the backend directory

```bash
  cd pogodex/backend
```

Install dependencies

```bash
  npm install
```

Start the backend

```bash
  npm run start
```
